API Information

The api endpoint is at /recipes

Parameters:

  • name - finds recipes with this exact entire phrase
  • searchterm - finds recipes with titles that contain this phrase
  • ingredients - for searching for recipes with certain ingredients. Seperate them with commas!
  • limit - limits the number of results sent back. The default is no limit.
All of the parameters can be used at the same time, but I using the name parameter
at the same time as the other parameters would be redundant, since at this time
every recipe is required to have a unique name, so the name parameter will always come up
with just one recipe.
